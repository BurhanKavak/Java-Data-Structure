### Java HashMap

#### Map içindeki sıralama, koleksiyon üzerinde tanımlanan iteratörün öğeleri bize veriş sırasıdır. Ancak HashMap yapısında sıra belirli değildir.


` map.put("Bilim", "12"); `<br>
`map.put("Sanat", "3");  `<br>
`map.put("Edebiyat", "5"); `<br>
`map.put("Siyaset", "9"); `<br>
`map.put("Java", "155"); `
